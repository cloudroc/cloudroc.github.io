又是美剧更新季，按照前两年的经验这应该是每年最快乐和充实的时间段。

这季虽然继续让戈登当主角，但是加入了很多老爷日后强劲对手发迹的描写，小丑的前身和谜语人尤其的刻画入骨。

磁力链接magnet:?xt=urn:btih:与迅雷thunder://地址进行转换

发表于 2015-10-19   |   分类于 代码   |  

在研究DHT，发现现在很多bt搜索的网站在放上磁力链接和种子的同时，又加上了一个thunder的地址。

磁力链接的获取好说，用海外主机伪装DHT节点使劲爬就可以，但是这个迅雷地址怎么来的，很无解啊…

于是搜索了好久，才发现这个迅雷地址是用磁力地址加密来的，直接看代码吧：

# -*- coding: utf-8 -*-
import base64

def thunderEncode(url):
    return ''.join(['thunder://',base64.b64encode(''.join(['AA',url,'ZZ']))])

# using example
if __name__ == "__main__":
    tUrl = thunderEncode("magnet:?xt=urn:btih:d9f48f4ddcccdd14d27c75977472eac905847173")
    print (tUrl)

AA ZZ怎来的，我也不知道，但是地址是对的，不信你试试

炒酸奶

发表于 2015-09-27   |   分类于 啰B嗦   |  

这几天成都依旧阴天，每天都会有小雨，貌似是不欢迎我。

前天晚上出去吃了一次成都的烧烤。因为本来晚饭就吃的很多，所以没怎么吃，但是喝了一肚子酒。然后又去了一个小酒吧，喝的德国黑啤酒，碰见两个年长的姐姐，聊得很有意思。就是混着喝酒回去头很疼。

于是昨天睡了一天，吃了同屋小伙的炝锅面。这小伙是直接把油热了热，就煮面了，葱蒜鸡蛋其实都买了，也难为他上了这么多年研究生。

今天是中秋。貌似很多年的中秋都是在外面过的，今年也不例外。下午去看了港囧，很多地方很搞笑，也让我想起了以前的一些事情，家庭依旧是个遥远的概念。

回来的时候吃了份炒酸奶，就是平时喝的比较浓的那种酸奶“炒”的。炒酸奶的是个东北妹儿，聊了一会，走的时候把钢勺子都送给了我。

 

 

宽窄巷子

发表于 2015-09-20   |   分类于 啰B嗦   |  

最近烟又抽多了，没有戒断的征兆。

来了趟成都。

趁周六他们都去参加婚礼，所以取消加班。正大光明的起身出去转了一遭。本来想去武侯祠，听了别人的建议就去了趟宽窄巷子。照例自己去的，所以此行比较无聊。买了天价的小吃（10元3个的春卷）还有一杯冰米酒，逛了一圈。因为地方比较小，所以又去了附近的人民公园，里面矗立着一个辛亥革命的纪念碑。

最后买了两个鲜无花果，觉得还是晒干的更喜欢一些。

本来成都是个吃货的天堂，不过最近没有食欲，加上真的比较辣，吃的不多。

还好宿舍可以自己做饭，能吃点自己喜欢的挂面+荷包蛋。

晚上尝试做了酱油炒饭，因为中午外面的米饭做的不错，所以味道还是不错的。

三体

发表于 2015-08-28   |   分类于 博客   |  

最近三体获了雨果奖，真是挺不容易的，虽然大刘挖了很多坑，收尾又很仓促都没填平，但是不影响他的宏伟框架的构成。

花了几天时间重新读了三体III。里面有句话说的很好：无知和弱小不是生存的障碍，傲慢才是。不过三体人和地球人都犯了傲慢的错误，三体人的傲慢，指的轻视地球人，认为他们不可能领悟黑暗森林原理，导致三体人被拽入威慑纪元。地球人的傲慢，指的云天明哥哥费尽心机编了三个童话，希望地球人类早点开发曲率飞船逃生，结果被误解为掩体计划，中间又被程心女神的人性光辉或者说爱，耽误了35年，结果直接导致太阳系被二维化。

最近还看完了徐中约的中国近代史，是一部实事求是的史书。尤其是现代部分，写的很真实，大不同于官方编写的史材。我把里面很多犀利的地方都加了标注和笔记，希望同步到z.cn的时候不会被出卖。里面很多地方可以和上面那句话印证。比如大清的以夷制夷和变法的失败，大约就是抱有傲慢的态度导致失败。再比如日本的战败，再到古巴导弹危机，再到后来的“一切反动派都是纸老虎”，如果能正视自己的国家矛盾或者正视自身组织不至于此，因为那样才能做出合理正确的决定，毕竟我们的时间线只有一个维度，只有正确的选择才有正确的结果，这也是悲剧产生的根源。
