web抓包工具Betwixt

发表于 2015-12-07   |   分类于 教程   |  

如果你觉得chrome自带的network抓包工具很好很强大的话，那么这个工具让你可以在本机全局层面使用类似chrome devtools的界面来分析流量了

其实git里面写的很详细，git地址：https://github.com/kdzwinel/betwixt

BTW,作者弄得那个动态图也很炫酷，一下就会被吸引住

简单说下：
1.下载源码
2.安装nodejs
3.命令行到安装目录，运行npm install 接着运行npm start，然后就可以用啦
4.如果chrome没有被goagent或者shadowsocket接管的话，需要设置一下端口代理，这个软件默认的是8008

设置方法：IE->设置->Internet选项->连接->局域网设置
->按照下图设置

然后就能看到效果啦

西行漫记

发表于 2015-11-15   |   分类于 啰B嗦   |  

这篇当然不是讲我西行或者西游记啦

最近辐射4上市，顺便玩完了辐射3，大呼神作！

虽然是09年的游戏，但是自己没玩过这类自由度非常高的RPG游戏。顶多就是玩过仙剑4，但是地图局限性很大，并且只能走规定的线路。而且剧情单一，顶多就是隐藏剧情。玩过之后和看电影一样，也是中国和日本游戏的局限性了。

辐射3完全不同，根据人物的选择可以触发不同的剧情造成不同的后果，可能是美国人信奉的actions have consequence。

另外一点就是地图巨大，一路下来，可能碰见各种NPC还有各种废土生物，你可以一路积德行善或者一路杀伐果断，甚至可以杀NPC。

除了广袤的废土，还有各种地图和山洞、地铁，里面危险重重，起初是自己孤孤单单，后面慢慢有了小弟，路上显得不是那么寂寞，这让我想起了西游记…

值得吐槽的是，里面虽然设定中国对美帝实施了核进攻，但是中国军官的指挥刀竟然是“大宝剑”，而且还有老式驳壳枪。

磁力链接magnet:?xt=urn:btih:与迅雷thunder://地址进行转换

发表于 2015-10-19   |   分类于 代码   |  

在研究DHT，发现现在很多bt搜索的网站在放上磁力链接和种子的同时，又加上了一个thunder的地址。

磁力链接的获取好说，用海外主机伪装DHT节点使劲爬就可以，但是这个迅雷地址怎么来的，很无解啊…

于是搜索了好久，才发现这个迅雷地址是用磁力地址加密来的，直接看代码吧：

# -*- coding: utf-8 -*-
import base64

def thunderEncode(url):
    return ''.join(['thunder://',base64.b64encode(''.join(['AA',url,'ZZ']))])

# using example
if __name__ == "__main__":
    tUrl = thunderEncode("magnet:?xt=urn:btih:d9f48f4ddcccdd14d27c75977472eac905847173")
    print (tUrl)

AA ZZ怎来的，我也不知道，但是地址是对的，不信你试试

数据sample：

磁力链接 ：magnet:?xt=urn:btih:d9f48f4ddcccdd14d27c75977472eac905847173

迅雷链接：thunder://QUFtYWduZXQ6P3h0PXVybjpidGloOmQ5ZjQ4ZjRkZGNjY2RkMTRkMjdjNzU5Nzc0NzJlYWM5MDU4NDcxNzNaWg==

炒酸奶

发表于 2015-09-27   |   分类于 啰B嗦   |  

这几天成都依旧阴天，每天都会有小雨，貌似是不欢迎我。

前天晚上出去吃了一次成都的烧烤。因为本来晚饭就吃的很多，所以没怎么吃，但是喝了一肚子酒。然后又去了一个小酒吧，喝的德国黑啤酒，碰见两个年长的姐姐，聊得很有意思。就是混着喝酒回去头很疼。

于是昨天睡了一天，吃了同屋小伙的炝锅面。这小伙是直接把油热了热，就煮面了，葱蒜鸡蛋其实都买了，也难为他上了这么多年研究生。

今天是中秋。貌似很多年的中秋都是在外面过的，今年也不例外。下午去看了港囧，很多地方很搞笑，也让我想起了以前的一些事情，家庭依旧是个遥远的概念。

回来的时候吃了份炒酸奶，就是平时喝的比较浓的那种酸奶“炒”的。炒酸奶的是个东北妹儿，聊了一会，走的时候把钢勺子都送给了我。
