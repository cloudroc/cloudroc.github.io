宽窄巷子

发表于 2015-09-20 | 分类于 啰B嗦

最近烟又抽多了，没有戒断的征兆。

来了趟成都。

趁周六他们都去参加婚礼，所以取消加班。正大光明的起身出去转了一遭。本来想去武侯祠，听了别人的建议就去了趟宽窄巷子。照例自己去的，所以此行比较无聊。买了天价的小吃（10元3个的春卷）还有一杯冰米酒，逛了一圈。因为地方比较小，所以又去了附近的人民公园，里面矗立着一个辛亥革命的纪念碑。

最后买了两个鲜无花果，觉得还是晒干的更喜欢一些。

本来成都是个吃货的天堂，不过最近没有食欲，加上真的比较辣，吃的不多。

还好宿舍可以自己做饭，能吃点自己喜欢的挂面+荷包蛋。

晚上尝试做了酱油炒饭，因为中午外面的米饭做的不错，所以味道还是不错的。

深夜食堂2.0

发表于 2015-08-30 | 分类于 啰B嗦

最近没空回家一直泡在封闭的地方，虽然是被封闭但是也没什么，就是时间久了无法回家觉得很窝火。主要体现在皮肤瘙痒起红包，并且不只我一个。貌似最近下雨比较多，大家招了某种床上的小虫子，而且这里没有洗衣机和大盆，只能靠人工晒被子…

之前说了kindle的事情，做了咬人画的的大合集，收录他最近几年所有的博客和漫画。但是咬人的作品还是比较少，只用了几天就看完了…

但是燃起了看漫画的兴趣，加上看了一点深夜食堂的电视剧，于是又简单写了个程序来抓深夜食堂原版漫画。不得不说，电视剧确实在某些地方超越了漫画，尤其是老板的刻画，多了很多血肉。

有了上次的经验，这次是针对pw2或者pw3的718px*965px大小的屏幕进行pdf生成的，对dpi和图片缩放都有了一定了解。

以下是放到pw3的实际效果，看着还不错哈，另放上一份sample供大家品尝：链接: http://pan.baidu.com/s/1nt062Vn 密码: 6ppm

 

三体

发表于 2015-08-28 | 分类于 博客

最近三体获了雨果奖，真是挺不容易的，虽然大刘挖了很多坑，收尾又很仓促都没填平，但是不影响他的宏伟框架的构成。

花了几天时间重新读了三体III。里面有句话说的很好：无知和弱小不是生存的障碍，傲慢才是。不过三体人和地球人都犯了傲慢的错误，三体人的傲慢，指的轻视地球人，认为他们不可能领悟黑暗森林原理，导致三体人被拽入威慑纪元。地球人的傲慢，指的云天明哥哥费尽心机编了三个童话，希望地球人类早点开发曲率飞船逃生，结果被误解为掩体计划，中间又被程心女神的人性光辉或者说爱，耽误了35年，结果直接导致太阳系被二维化。

最近还看完了徐中约的中国近代史，是一部实事求是的史书。尤其是现代部分，写的很真实，大不同于官方编写的史材。我把里面很多犀利的地方都加了标注和笔记，希望同步到z.cn的时候不会被出卖。里面很多地方可以和上面那句话印证。比如大清的以夷制夷和变法的失败，大约就是抱有傲慢的态度导致失败。再比如日本的战败，再到古巴导弹危机，再到后来的“一切反动派都是纸老虎”，如果能正视自己的国家矛盾或者正视自身组织不至于此，因为那样才能做出合理正确的决定，毕竟我们的时间线只有一个维度，只有正确的选择才有正确的结果，这也是悲剧产生的根源。

泪奔九英里

发表于 2015-08-12 | 分类于 代码 ， 技术宅

上篇说了新近一发kindle,以博主多年技术宅的身份,不免要捯饬一下。

于是有个idea，平时经常看咬人画的里面的宅人食谱，所以想干脆把咬人大神的博客合订一下做成pdf，放到kindle上面看。

到今天晚上终于成功了，附上kindle效果图：

缺点：
字体还需要调整一下，咬人大神用色出神入化，放到kindle这种电子墨水屏幕不免丧失原图的用色美（但是pdf保留了色彩原图），还有就是咬人画了很多好看的大尺寸壁纸，用色鲜明艳丽，奈何pdf有尺寸限制，所以无法显示全面，放到墨水屏看更是掉渣。

目前做了2014年的合订本，作为睡前必读物。

技术这里先不讨论，使用的是jsoup+itext，前者解析网页结构，后者用于生成pdf。

今年写的唯一较为实用的程序，且放到kindle中宅人食谱画质感人，博主已泪奔九英里……

深夜食堂&桃花期

发表于 2015-08-06 | 分类于 啰B嗦

最近没写有意义的代码，依旧更新口水文。

看了一部日本电影叫深夜食堂，是根据电视剧拍的电影版，但是我没有看过电视剧，看完觉得虽然有的地方很啰嗦但是觉得整体不错，有了看电视剧的兴趣。给我同样看完电影想看电视剧感觉的还有桃花期Moteki。

两部影片虽然塑造了不同个性的人物，但是叙事风格类似，都是一个个小故事拼接起来的。不像诺兰那种碎片话叙事，完全体现的是导演的个人能力，这种顺序叙事则完全靠故事情节的引人入胜还有演员到位的阐释。

深夜食堂食堂里面的刀疤大叔做的一手好料理，尤其是里面的鸡蛋卷和铁板意面还有咖喱饭（虽然简简单单但是看的我食指大动），都是深夜这个神秘时间段治愈小人物心灵创伤的神奇药物。功效应该和TVB的“我去给你下碗面”类似，其实主要是做饭的人还有陪你吃饭的人，日本料理或者是方便面都是药饵而已。需要强调的是，这位大叔一般作为一个陪同者，和小人物们一起感受经历，没有强加自己任何想法。写到这里想起桃花期电视剧最后一集藤本老爹和藤本幸世说的一句话：爸爸和妈妈不想看到你活得不快乐的样子，随你吧，是你的人生。如果有想做的事情就去做，没有的话就去找…
