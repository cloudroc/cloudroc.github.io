相逢何必曾相识

发表于 2014-04-07 | 分类于 博客

最近公司竞聘储备人才，散落各地的项目同事纷纷赶回来了。于是吃饭喝酒在所难免，今晚很荣幸的蹭了一顿饭，觥筹交错了几回认识了好几个常年没有见过的老前辈。还见到了阔别好久的聂姐，三年不见，孩子都已经五岁了，她还是看着那么年轻。

散席后和小陈总去了他住的宾馆，他给我看了几个相集装订的很好有的还写明了标价，拍的都很好，扉页还写好了赠言，说是一个叔叔给的。说了一会，原来这个叔叔是公司大领导的父亲，俩人在边疆摄影相识，结为忘年交。因为得知他是回来竞聘的还想特地关照他一把…

回来后想了想，何必计较是否真的非要帮上忙呢，能相遇都是缘分，难得的其实是交流时候能得到的共鸣。不管这种悸动是单方向的还是双向的。

装修房子尚未开工，相亲的行动已慢慢开始了。见了几个，都请她们吃饭聊天什么的，但是都没有引起内心的悸动，可能眼光真的很挑剔，当然自己条件不见得很好，但是毕竟是讨她来要做一辈子的老婆，也不想草草就找了，但是这很得罪好心给介绍的那些朋友，非常纠结。

这些天经常梦见一个人，相逢何必曾相识。

滇池海鸥、连城诀etc

发表于 2014-03-17 | 分类于 博客

最近依然在昆明，经历了301惨案，除了两个大学时代要好的哥们，父母亲友都没有打过电话。那时候，人生的寂寥感分分钟就上来了。

后来的一段时间，系统运行上线，很多事情忙不过来，也没有出去玩。照这个忙样，去大理的计划估计也要泡汤了~

每天睡前读小说的习惯，也因为累的原因放弃了一段时间，但是最近又拾起来了，把连城诀读完了。十分喜欢血刀老祖这个角色，这个人坏到极点心但思缜密又十分讲江湖原则，哪怕知道自己被困，也想把虏来的水笙强暴，免得留下遗憾。然后就是丁典，绝对是扫地僧级别的人物了，虽然被穿了琵琶骨，又被挑断脚筋，但是凭着神照功还是分分钟灭了血刀门和雪山派的高手，绝对是所谓的出场就满级的人物。尤其是雪山派的那个人还穿着乌蚕衣，说明是雪山派的顶尖人物，应该和侠客行的白万剑差不多吧，被丁典轻易推到了，可想这家伙多厉害了。

书中暴露了很多人性的黑暗面，花铁杆可以把自己情同手足的兄弟尸体拿来充饥，可以说是实实在在的被打的“落花流水”了，万震山师兄弟、凌退思等人的描写，均是衣冠禽兽，最终都死在宝藏的毒药上了。

但是回头想想自己，如果自己也有那么大能力，也有机会获得富可敌国的大宝藏，是不是亦是如此呢。

上周六，抽空去看了海鸥，了了自己一直的一个愿望，之前在这里一年半，都没见过它们。据说3月底它们就要飞去西伯利亚了，抓紧拿起手中的nexus 5拍上几张。

满城春色 双手合十

发表于 2014-02-16 | 分类于 博客

最近不幸又被发配到了云南，想想之前曾经有1年半的时间都在这里度过了，那时候迷迷茫茫，傻傻忽忽，一心就知道干活，没有现在那么多烦心事，呵呵…

来了差不多一周的时间，其实真正干活的时间也就两三天，大部分时间基本在混日子。其实不忙也不好，比如情人元宵节这种喜大奔庆的节日，终究带着强烈的落寞感在不能叫为家乡的地方度过了。

于是今天虽然领导强烈要求加班，我还是软磨硬泡的请假跑到了圆通寺烧香。

坐着A2到了圆通山站下车，虽然圆通山很明显就看到了，但是遍地寻不到圆通寺的门口。看到路边有个做益达促销的小姑娘，就跑过去问路，小姑娘估计也是外地人，竟然用标准的普通话大概指了指方向，说自己也不知道具体的地点，然后友好的冲我笑了笑，继续低下头玩手机……即使这样，心中也是暗喜啊，毕竟来到昆明之后第一次有机会和女性交谈，平时说话的都是老爷们……

渐渐的看到路边越来越多卖香火贡品和八字算命的，很快就找到了地方

门票只要六块钱，进去之后有免费的环保香和蜡烛，这下倒是省下香火钱了。进去之后马上就被众多信徒的虔诚所打动。印象最深的就是一个老婆婆带着自己的孙女在千手观音像前面虔诚的叩拜，婆婆嘴里用昆明话默念着什么，孙女非常认真的跟着一块叩拜，拜完后一块放上准备好的糕点贡品。这些都加深了自己对寺庙的畏惧感。

中午在寺庙里面花了五元吃了一顿全素的午餐，不知道用的什么油，反正菜做出了就是菜的味道，十分香甜，虽然中间加了一次饭，但是吃完之后还是马上就饿了……

满城春色，双手合十，默念自己的愿望吧。



读曾国藩家书

发表于 2014-02-06 | 分类于 博客

年关无事,无心研究技术,奈何总是被问起有没有对象的事情,所以初二之后一直宅居在家,除了见了失散多年的文超同学,其他交际几乎为0…

期间翻阅了曾国藩家书,书是文言白话的,也就是曾国藩怎么写的,书上就是怎么记录的,没有白话翻译,区别在于加上了标点符号便于现代人阅读.

家书写的都不是很长,基本千字之内,结合自身粗鄙的历史知识,大概能还原书信原有的意思吧.

书信大概分了这么几种:写个祖父祖母请安的,写个父亲母亲请安的,写个弟弟和其他兄弟的交代家中大小事情的,写个自己俩个儿子的,还有附录写个家中妇女的教导他们努力从事农业生产的(这个应该不是书信).其中大部分还交代了自己当时的征讨”太平天国”,”捻军”的战斗情况.

前面几块写个自己长辈和弟弟的书信看的很粗略,不想细看,因为感觉没啥意思,再就是自己阅历不够,责任心不够等等原因…

看到写给曾纪泽、曾纪鸿的书信，其中主要是写给曾纪泽，感觉自己一直在被教育，以下两段摘抄下来了，感觉受教很深.

一段是讲自己戒烟来教育儿子锻炼意志:

人之气质，由于天生，本难改变，唯读书则可变化气质。古之精相法者，并言读书可以变换骨相。欲求变之之法，总须先立坚卓之志。则以余平言之，三十岁前，最好吃烟，片刻不离，至道光壬寅十一月二十一日立志戒烟，至今不再吃；四十六岁以前做事无恒，近五年深以为戒，现在大小事均尚有恒。即此二端，可见无事不可变也。尔于厚重儿子，须立志变改。古称“金丹换骨”，余谓立志即丹也。此嘱。

一段讲如何练字

以后每日习柳字摆个，单日以生纸临之，双日以油纸摹之。临帖宜徐，摹帖宜疾，专学其开张之处。数月之后，手愈拙，字愈丑，意兴愈低，所谓困也。困是切莫间断，熬过此关，便可少进；再进再困，再熬再奋，自有恒通精进之日。不特习字，凡事皆有极困极难之时，打的通的，便是好汉。
