与2345签到共同成长

发表于 2015-05-01 | 分类于 代码

刚写下这个标题就有点觉得不科学，其实应该是和2345签到共同变老吧。

整整一年多时间没有发代码，去年也是太累，今年没有好转的迹象，应该会更累的，然后就是慢慢变老……

2345签到一直在改，估计是发现有人自动签到，所以加了很多限制，从他们的js发现很多判定模拟登录的脚本了。但是…其实水平还是有限，无法做到淘宝那么牛逼啊，仔细分析下抓包还有脚本还是能发现些蛛丝马迹，完成程序自动签到的。BTW，他们的js代码好多注释，省去很多弯路哈

本来是用java的httpclient做的，并且想放到sae上面，但是发现本地运行好好的，放sae就有问题，尝试建立cookie的存储还是不行，云空间什么的有点无力。然后试了试sae的python，发现竟然python可以完美的模拟登录，但是python水平实在太弱，就有了下面的代码：

#新版2345自动签到
@app.route('/2345checkin1' , method='GET')
def checkin():
    username = "xxxx@126.com"
    password = "自己把密码md5一下"
    cmd = "login"
    weakpass = "0"
    vTime = "抓包获取"
    weakpass = "0"
    showWay = "ajax"
    forward = "http://jifen.2345.com/m/login.php"

    # Init
    opener = urllib2.build_opener(urllib2.HTTPCookieProcessor(cookielib.CookieJar()))
    urllib2.install_opener(opener)

    # Login登录
    login_url = 'http://jifen.2345.com/jifenLogin.php'
    login_data = urllib.urlencode({ 'cmd':cmd,'forward':forward,'password':password,'username':username,'vTime':vTime,'weakpass':weakpass,'showWay':showWay})
    login_headers = {'Referer':'http://jifen.2345.com/', 'User-Agent':'Mozilla/5.0 (Windows NT 6.2) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/27.0.1453.116 Safari/537.36 2345chrome v2.3.0.2200'}
    login_request = urllib2.Request(login_url, login_data, login_headers)
    login_response = urllib2.urlopen(login_request).read()

    #get md5code
    index_url = "http://jifen.2345.com/index.php"
    index_request = urllib2.Request(index_url,None , login_headers)
    index_response = urllib2.urlopen(index_request).read()

    #pattern这段主要是官方为了防止模拟领取写的脚本验证了
    u_pattern = re.compile(r'var u = "(.*)";')
    u_result = u_pattern.search(index_response)
    u = u_result.group(1)

    code_pattern = re.compile(r'var code = "(.*)";')
    code_result = code_pattern.search(index_response)
    code = code_result.group(1)

    token = md5(u+code)

    #checkin签到
    chekin_url = "http://jifen.2345.com/jifen/every_day_signature_new.php"
    checkin_data = urllib.urlencode({ 'sign_token':token})
    checkin_headers = {'Referer':'http://jifen.2345.com/', 'User-Agent':'Mozilla/5.0 (Windows NT 6.2) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/27.0.1453.116 Safari/537.36 2345chrome v2.3.0.2200','X-Requested-With':'XMLHttpRequest'}
    checkin_request = urllib2.Request(chekin_url, checkin_data, checkin_headers)
    checkin_response = urllib2.urlopen(checkin_request).read()

    #buy
    buy_url = "http://buy.2345.com/sign_up/"
    buy_headers = {'Referer':'http://buy.2345.com/?sign=1', 'User-Agent':'Mozilla/5.0 (Windows NT 6.2) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/27.0.1453.116 Safari/537.36 2345chrome v2.3.0.2200','X-Requested-With':'XMLHttpRequest'}
    buy_request = urllib2.Request(buy_url, None, buy_headers)
    buy_response = urllib2.urlopen(buy_request).read()

    return buy_response

完成的事情：
1、自动登录
2、自动签到
3、自动领取buy.2345.com的签到金币，据说可以提现的

黑暗骑士

发表于 2015-02-17 | 分类于 博客

一口气看完了《蝙蝠侠：第一年》《黑暗骑士归来》上下，以上皆为动画版。才发现作为一只年近三十每日挤公交不修边幅蹲坑玩手机每天保持坐姿14小时以上的单身狗，我依旧那么喜欢蝙蝠侠。

DC和marvel都塑造了很多英雄，marvel的作品偏重于现代的科技感和各种神秘的概念和世界观，但是英雄缺乏性格，美队是个例外，除了美队，marvel的其他英雄在人格魅力上都过于无力了…相比之下DC的大超和老爷都显得十分有人格魅力，一个是光明之子一个是黑暗骑士，但是相比高大上的人物，现在的我们更喜欢平凡神秘的bruce老爷，不信请看诺兰的黑暗骑士三部曲，最近的蝙蝠侠衍生剧哥谭市，以及DC近年的蝙蝠侠动漫。

黑暗骑士归来上下里面的老爷已经十分年迈，但是塑造的蝙蝠侠形象已经超越了诺兰的黑暗骑士（个人观点而已），并且把美国政府的官僚化和所谓的正义诠释的很好，最经典的就是关于蝙蝠侠的处理总统交给州长，州长又交给市长，市长又把皮球踢给了GCPD局长，最后大超又作为美国版正义的化身要去和老爷清算，被老爷揍得抬不起头，哈哈。这也说明了为什么蝙蝠侠要作为一个象征，潜伏于黑暗中。但是这个黑暗骑士又这么光辉伟岸，单靠领导力就拯救哥谭市民于混乱之中，使得连续两代GCPD局长的评价都是：He’s too Big.注意这里不是器大活好的意思…

尤其在下部里面老爷终于完成了宿命的对决，一次是和小丑的一次是和大超的。和小丑的对决中，相比之前的不使用手枪和绝不杀人的蝙蝠侠，这里老爷被迫使用了手枪并且最终把小丑制服，虽然最后是小丑自己在全身瘫痪的情况下拧断了自己的脖子，科学性有待研究。虽然杀了小丑并不改变什么，但是至少可以拯救很多很多其他人的绳命。这又说明，老爷最后终于想明白了，不去维护美帝那套虚伪的民主法制体系，仅仅作为黑暗中的明灯就好。

最后管家阿尔弗雷德在寒冷的风雪中死去。

今年夏天一定得买件蝙蝠侠的T恤。

在景洪的最后一天

发表于 2015-01-26 | 分类于 博客

景洪是其实就是西双版纳的州政府所在地.

24号是在版纳的最后一天，在边疆匆匆忙忙的干了两个月的活，除了去总佛寺和曼听公园，只去过原始森林谷。原始森林谷景色还不错，里面有科目人，动不动就“哈呼”一下吓唬你。曼听公园里面有大象表演，但是没什么可看的，总佛寺最为不错，是东南亚小成佛教的总写，就在曼听公园的后面而且不要门票，后来只要是周末无事就去膜拜一次。

最后一天就约一个同事去了基诺山，联系好客户找了一个本地人，可以买到特价票。

开始有点不爽。打了一个黑车，个人觉得是个无耻的人，说好包车一天200，其实他能赚很多了。中间听说我们自己买票，他就开始闹了，因为要是他能给我们买票买票他能赚更多的，还要强迫去珠宝店云云，我这种外表老实的人都要发作了。一下车就要我先给他200，我只给了100，中间联系熟人的时候，这个家伙如跗骨之蛆一般跟着，因为没有马上联系到熟人，他就又开始闹了，希望要够200之后回去，于是纠缠一番，一块没有多给，就叫他速速滚粗了…

然后熟人来了，其实是客户的一个客户，和销售处的人说了一阵我听不懂的语言，就帮买到了特价票，还不到零售票价的三分之一。然后骑摩托带我们到了附近的村寨。他普通话非常好，聊天之间，才得知他就是基诺族，是56个民族的最后一个民族，刚才买票时候说的话就是他们自己的语言了。中间得知他还自己做茶，自己搭了一个木头顶棚专门晒茶用的。品尝了他家的生普洱还有滇红，都是好货，和昆明专卖店里面的普洱一个味道。于是买了一公斤滇红，才100块钱，是友情价了，和同事各拿一斤。要知道我在昆明买一盒滇红，估计不到3两就要40块钱了。

中午去村寨附近吃了顿午饭，和他可爱的女儿一起，聊到很多他的见闻和边疆的事情。

吃完后又带我们来到基诺山景区门口，还帮请到一个导游，属于免费了，感觉很感动，又不知道怎么感谢，但又草草的道别了。

导游很帅，同行也有不少美女，讲讲解了很多基诺人的事情，基诺就是舅舅的孩子的意思，因为他们之前都是近亲结婚的，家里舅舅地位最高。他们的女神叫阿嬷基北，就是大地女神的意思。景区很小，更像一个大山寨了，中间看了基诺族的表演，发现原来基诺族的姑娘更美，比起之前见得彝族哈尼族都漂亮。

闲聊碰到一对苏州开车来的夫妻，我这位同事很健谈，很快都聊得很开心，于是就问能不能回去的时候一块，他们欣然答应了。

回去的时候又和这对夫妻聊了很久，女的说话软软的，的确是苏杭那边的人。下车的时候，我想给他们车费，但是被拒绝了，那位大哥说都是萍水相逢，这样就不好了，又是一阵感动。

景洪再见。
