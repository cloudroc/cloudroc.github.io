新年了换个编辑器-Sublime Text 2，程序猿必备神器！

发表于 2013-02-06   |   分类于 技术宅 ， 教程   |  

看到好多牛X的人都在用Sublime Text这种神器，于是赶紧下载来体验一下吧。

之前一直用Notepad++，因为能集成SVN插件，用起来还是不错的。又因为我的Notepad++装在的“云端”上面了，之前想改过主题，总是以失败告终，又懒得再下载一个绿色版的，干脆现在用Sublime Text吧，因为看起来很专业啊~

关于 Sublime Text 2：

是一款收费的商业软件，个人授权费高达 59 美刀。
虽然是付费软件，但是作者真是人品第一，竟然让大家无限制免费试用，只是时不时会弹出付费的窗口提醒。不过最新的2.0.1是可以破解的，因为我亲测成功了，破解方法在后面。

1、下载安装和汉化

Sublime Text官网地址：http://www.sublimetext.com/（官方首页有个gif的图片，看到后你是不是已经动心了？）

按自己的系统下载对应的安装包就可以，我下载的是 Windows portable version

汉化方法（转自isaced博客）

汉化包下载地址

以目前最新版Sublime Text 2为例，安装好该软件后，下载汉化包，解压到系统盘中Sublime Text 2的”packages/Default”目录下，覆盖原始文件，无需重启，Sublime Text 2菜单栏就汉化完成了。

提示：

(1)Packages路径，可通过 Sublime Text 菜单中的 Preferences > Browse Packages 找到 Packages 目录

(2)可在 Sublime Text 安装目录下，手动创建一个Data目录,然后重启 Sublime Text 会自动在Data目录下生成Packages目录，之后其它步骤同上

PS:因为平时英文的IDE用的比较多，汉化之后感觉各种别扭，就换到英文的了

2、破解方法

所用工具：十六进制编辑器，如WinHex

(1)安装最新版本的Sublime text 2.0.1版本。备份并复制一份sublime_text.exe文件

(2)用WinHex十六进制工具打开备份的文件，跳到000CBB70行，将该行的8A C3改为B0 01保存即可

然后看help下面的about，就看到这个窗口了，提示 Registered to Unlimited User License ，hoho~

3、安装Package Control插件

Package Control组件是Sublime Text 2的插件管理器，安装插件管理器也很方便！

按Ctrl+~调出控制台，粘贴如下代码到控制台

4、把将Sublime Text2 加入右键菜单

  1. 运行中输入 regedit 打开注册表

2. 在HKEY_CLASSES_ROOT/*/shell/ 下新建’项’ ,名称自己觉得.我用的是Sublime Text

3. 在 Sublime Text 下 新建’项’ 名字:command (这个貌似不可以修改)

4. 点击 command ,在右边的(默认),双击填入下面的值:

D:Program FilesSublime Text 2.0.1sublime_text.exe “%1”

前面是exe的绝对物理地址, %1 用tab方式打开不同的文档(必须)

 

5、配置python编译环境（转自isaced博客）

打开 SubLime Text 的 Packages 文件夹，里面有一个 Python 文件夹，编辑里面 Python.sublime-build 这个文件，最简单的方法是复制以下内容替换，网上有很多帖子说的是cmd第一个参数为python，但是这样需要手动添加一个系统环境变量，所以说直接把这个参数替换为你的Python目录地址更为方便。

{
    “cmd”: [“C:\Python33\python.exe”, “-u”, “$file”],
    “file_regex”: “^[ ]File “(…?)”, line ([0-9]*)”,
    “selector”: “source.python”
}

如果你想使用环境变量那么就把cmd第一个参数改为python，然后在环境变量中的path中添加Python的路径，如：“C:Python33”

然后我们新建一个文件后缀为.py，Build Sustem 选择默认的Automatic就可以了，编完码保存选择菜单中的 Tools -> Build (或者直接Ctrl+B) 就可以build了。


ps：这种方式不支持input（）这种函数的，算是有点缺陷吧~

话说小黄鸡被封或者说限制有一段时间了。大概是每个ip只能向官方请求几千次，棒子的服务器就认为你恶意调用了，就会返回

Unauthorized access!. In this program(site, app), the SimSimi API is being used illegally. Please contact us. http://developer.simsimi.com
然后呢，然后你就没得玩了。

我看有的朋友之前收集了不少词库，这次被封也不是很怕，之前利用分词调用之前的词库就行了。但是我当时根本没弄词库这回事啊~就不骂自己了吧

萌生了个想法，是不是可以调用人人上和小黄鸡的聊天呢？人家是有棒子官方key的撒~

正好oschina上面有人发了如何用python模拟登陆人人网，传送门：Python登录人人网并抓取新鲜事

于是照葫芦画瓢，登陆之后加了一段调用小黄鸡对话的代码，发现果然可行，吼吼~
