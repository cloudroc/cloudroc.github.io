喜欢就买

发表于 2015-08-11   |   分类于 博客 ， 啰B嗦   |  

最近买了一个日版七代的kindle paper white，之前一直使用安卓上面的kindle应用，曾经带来不少快乐，但是后来总是闪退，刷机之后也忘记了安装。

这次直接买了一个pw3，果然不同凡响，阅读兴趣大增，用kindle看完了一直磨磨唧唧没读完的《永恒的终结》，对里面的因果链逻辑不做分析，就记住里面的口头禅“伟大的时间之神啊”。

今年真正做到了“喜欢就买，不行就分，多喝点水”的处事规则，下一步就差戒烟了。



很惭愧，只做了些微小的工作，您的支持将鼓励我继续努力创作！
